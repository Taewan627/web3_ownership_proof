# AI-Web3 소유권 증명 시스템

분석형 AI로 완성하는 데이터 주권의 진짜 증명

## 특징

- 🔐 **AI 기반 소유권 검증**: 머신러닝으로 월렛 소유권을 정확하게 분석
- 🌐 **온체인 데이터 분석**: 블록체인 트랜잭션 패턴을 심층적으로 분석
- 🛡️ **영지식 증명(ZKP)**: 개인정보를 노출하지 않고 소유권 증명
- 🆔 **DID 신원 인증**: 분산 신원 시스템과 생체 정보 연동
- 📊 **실시간 분석 로그**: 상세한 검증 과정을 실시간으로 확인

## 기술 스택

- **Frontend**: React 18, Tailwind CSS
- **Build Tool**: Vite
- **Icons**: Lucide React
- **Deployment**: Hugging Face Spaces

## 사용 방법

1. 지갑 주소를 입력합니다
2. '검증 시작' 버튼을 클릭합니다
3. AI가 실시간으로 소유권을 분석합니다
4. 상세한 분석 결과와 신뢰도 점수를 확인합니다
